senaiInputText
directive in module SenaiInputTextDirective

Description

Diretiva que permite criar um campo de texto no padrão bootstrap.

Usage

as element:
<senai-input-text
       ng-model="{expression}"
       preffix="{string}">
</senai-input-text>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelexpression

Modelo que será alterado pelo input.

preffixstring

Prefixo que ficará à esquerda do input.
